Es gibt 2 Wege die iTunes ID für einen Podast zu finden:

1. In der iTunes desktop Applikation

Öffne iTunes auf deinem Computer und gehe zur Seite des Podcasts. Am einfachsten geht das über die Suche. Auf der Podcast-Seite findet sich in der linken Spalte ein Button, der mit Abonnieren beschriftet ist. Geklickt werden muss auf den kleinen Pfeil am rechten Rand des Buttons. Dann gibt es eine Auswahl Link kopieren.

Diesen Link kopiert man dann in einen Text-Editor. Die eigentliche ID ist der letzte nur aus Zahlen bestehende Teil der URL (siehe Markierung im Bild unten).

2. Auf Apple's Podcasts Connect Plattform

Gehe auf https://podcastsconnect.apple.com und melde dich mit deiner Apple ID an. Klicke dann auf das Cover des in Frage kommenden Podcasts. Auf der folgenden Seite findest du die sogenannte Mirror URL. Die ID ist der letzte nur aus Zahlen bestehende Teil der URL (siehe Markierung im Bild unten).

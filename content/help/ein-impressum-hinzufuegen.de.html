--- title: Ein Impressum hinzufügen keywords: - blog - custom-pages aliases: ["/docs/ein-impressum-hinzufuegen"] ---

Um eine neue Seite auf der Website anzulegen muss zunächst ein benutzerdefiniertes Theme erstellt werden. Die Anleitung dazu findet sich hier: Ein benutzerdefiniertes Theme erstellen

Anschließend wird im letzten Tab des Theme-Editors eine neue Datei namens impressum.html angelegt.

Als Startpunkt für das Impressum gibt es unter diesem Absatz eine Vorlage, in der im Bestfall nur Name und Adresse ausgetauscht werden muss. Ein eigenes Impressum kann man zum Beispiel mit dem Generator von eRecht24 erstellen. In beiden Fällen muss der HTML Code kopiert und in die eben angelegte Datei eingefügt werden.

Zum Schluss muss die Datei nur noch gespeichert werden. Danach ist die neue Seite unter https://mein-podcast.podigee.io/pages/impressum (mein-podcast.podigee.io muss durch deine eigene Website-Domain ersetzt werden) verfügbar.

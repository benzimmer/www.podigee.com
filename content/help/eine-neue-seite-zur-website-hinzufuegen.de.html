--- title: Eine neue Seite zur Website hinzufügen keywords: - themes - custom-pages aliases: ["/docs/eine-neue-seite-zur-website-hinzufuegen"] ---

Um eine neue Seite auf der Website anzulegen muss zunächst ein benutzerdefiniertes Theme erstellt werden. Die Anleitung dazu findet sich hier: Ein benutzerdefiniertes Theme erstellen

Anschließend wird im letzten Tab des Theme-Editors eine neue Datei die den Namen der neuen Seite trägt angelegt. Der hier gewählte Dateiname bestimmt auch die Adresse unter der die Seite später verfügbar ist.

Als Beispiel wäre die Seite mit dem Namen impressum.html später unter https://mein-podcast.podigee.io/pages/impressum verfügbar. Ein etwas "extremeres" Beispiel wäre eine Datei names Meine-Tolle-Neue-Seite.html, die dann entsprechend unter https://mein-podcast.podigee.io/pages/Meine-Tolle-Neue-Seite zu erreichen ist.
